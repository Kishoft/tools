Data Types (Tipos de datos)

Hay varias maneras de representar información en JS

String (cadenas de texto)
  • "String"
  • 'String'
  • `String`
Number (Números) - Son Integers o Floating Points
  • 3
  • 3.234
  • 3e+2 (la e es de exponente)
BigInt (Gran entero) - Son Integers de precisión arbitraria (ocupan tanto espacio como tengas)
  • 792834732894723984723894n (Se les pone una "n" al final al momento de declararlos sin constructor)
Boolean - True o False, representan 2 estados
  • True
  • False
Undefined - Aparecen cuando una variable se inicia sin valor definido
let variable; //Devuelve undefined
Null - Se utiliza para dejar vacía una variable, constante o propiedad
let variable = null;
Infinity - Representa un número infinito, también puede ser negativo
let infinito = Infinity;
Symbol - Sus instancias son únicas e inmutables, similar a propiedades privadas de una clase
let variable = Symbol();

//Podemos usarlo para hacer propiedades privadas sin usar clases

let objeto = {
    [variable] : "World"
}

//Cuando llamemos a objeto[variable], devolverá un Symbol inmutable
Object - Conjunto de "par NO ordenado del tipo clave-valor" que representa una "instancia anónima", sus claves (keys) solo pueden ser "String" y "Symbol"
let objeto = {
    prop1 : 'String',
    prop2 : 1234,
    prop3 : true,
    prop4 : {},
    prop5 : []
}
Set - Conjunto de "valores únicos iterables SIN índice interno implícito", por lo tanto no garantiza conservar el orden
let set = new Set();
set.add(1);
set.add(1);
set.add(2);

//Solo almacena [1, 2], el valor repetido se descarta
Map - Conjunto de "par clave-valor iterable por orden de insersión de su contenido", similar a los Object con la diferencia que su clave (key) puede ser "String", "Number", "Object" o incluso NaN y se pueden iterar directamente
let map = new Map();

map.set("clave","valor");
Array - Conjunto de "valores repetibles e interables con índice implícito". Conservan el orden de insersión
let array = ['uno', 'dos', 'tres'];
