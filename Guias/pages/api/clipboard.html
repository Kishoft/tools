Clipboard API

Comprobar que se pueda utilizar

if ("Clipboard" in window) {
    /*Clipboard se puede utilizar*/
}
else {
    /* API no disponible */
}

Métodos

read()

Lee información arbitraria (como imágenes, por ejemplo) del portapapeles (clipboard)

readText()

Lee solo texto del portapapeles

write()

Escribe información arbitraria (como imágenes, por ejemplo) en el portapapeles (clipboard)

writeText()

Escribe solo texto en el portapapeles

Consumiendo la API

let copiedText = await navigator.clipboard.readText();
